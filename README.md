# 💰 Expense Tracker

A modern Expense Tracker built with HTML, CSS, and JavaScript to manage personal income and expenses. The application stores data in Local Storage, displays a financial summary, and visualizes spending using Chart.js.

## 🚀 Features

- Add new transactions
- Edit existing transactions
- Delete transactions
- Search transactions
- Filter by Income or Expense
- Transaction date support
- Automatic balance calculation
- Financial dashboard with Chart.js
- Local Storage support (data persists after refresh)
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
